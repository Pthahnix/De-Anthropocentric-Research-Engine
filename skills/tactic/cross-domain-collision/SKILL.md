# Cross-Domain Collision Tactic (Category 2)

## Minimum Collision Methods

<HARD-GATE>
**You MUST use at least 2 of the 4 collision methods:**
- facet-bisociation (MANDATORY — this is the core collision method)
- analogical-transfer
- random-paper-entry
- forced-bridge

facet-bisociation is always required. Choose at least 1 additional method.
More methods = more diverse cross-domain ideas.

The cross-domain facets extracted during ideation STEP 0 are the PRIMARY INPUT.
If no facets are available, this is a protocol error — go back to STEP 0.
</HARD-GATE>

## Procedure
1. If facets from multiple domains available: **facet-bisociation SOP** (Koestler bisociation)
2. If source domain identified: **analogical-transfer SOP** (Synectics)
3. Pick a random paper from the survey: **random-paper-entry SOP** (serendipity)
4. If two unrelated techniques exist: **forced-bridge SOP** (forced connections)
5. Aggregate hybrid ideas, prioritize by connection strength
