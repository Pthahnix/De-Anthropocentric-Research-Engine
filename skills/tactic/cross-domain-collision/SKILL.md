# Cross-Domain Collision Tactic (Category 2)

## Procedure
1. If facets from multiple domains available: **facet-bisociation SOP** (Koestler bisociation)
2. If source domain identified: **analogical-transfer SOP** (Synectics)
3. Pick a random paper from the survey: **random-paper-entry SOP** (serendipity)
4. If two unrelated techniques exist: **forced-bridge SOP** (forced connections)
5. Aggregate hybrid ideas, prioritize by connection strength
